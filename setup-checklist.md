- [ ] Emergency protocol defined

### Memory Files

- [ ] Services & pricing
- [ ] FAQs (20 common questions)
- [ ] Escalation rules

### Channels
- [ ] Telegram bot set up
- [ ] WhatsApp connected (if applicable)
- [ ] Web chat embedded (if applicable)
- [ ] Phone number configured (if applicable)
